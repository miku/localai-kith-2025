# Open models for AI applications

> [KI-Tage](https://ki-tage.webwirtschaft.net/) Halle (Saale), 2025-11-20/21, [90min]

## Abstract

Die meisten kennen KI-Modelle der großen Anbieter, die entweder über eine
Webseite, App, Desktopapplikation oder eine API nutzbar sind. Seit der
Veröffentlichung von [Llama](https://arxiv.org/abs/2302.13971), einem
Foundation Model von Meta, im Februar 2023, haben auch offene Modelle (oder
"open weights models") eine kontinuierliche Weiterentwicklung erfahren.

In dieser Session wollen wir Möglichkeiten und Grenzen offener Modelle anhand
einiger Beispiele illustrieren, oder um das Modell gpt-oss:20b zu Wort kommen
zu lassen:

> Seid bereit, eure Daten in euren eigenen Räumen zu halten, ohne dabei auf die
Leistungsfähigkeit eines großen Sprachmodells zu verzichten. Wir zeigen euch,
wie ihr das möglich macht, und welche Überraschungen dabei auf euch warten.
